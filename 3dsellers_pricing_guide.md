## How to Create a Rule in 3DSellers

1. **Go to:** Automation → Rules → Add New Rule

2. **Set Conditions:**
   - "Title contains" [keyword]
   - OR "Category equals" [your category]

3. **Set Action:**
   - "Change price" → "Increase by percentage" → [percentage]

4. **Set Schedule:**
   - Start Date: [X days before event]
   - End Date: [event date]

5. **Save and Activate**

---

## Pro Tips

- **Stack rules carefully** - Rules can compound, so be careful with overlapping dates
- **Test with one item first** - Apply to a single listing before bulk
- **Monitor performance** - Check if price increases affect sales velocity
- **Revert automatically** - Set end dates so prices return to normal
